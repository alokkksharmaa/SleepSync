SleepSync

A modern full-stack portal to track sleep patterns, analyze wellness insights, and provide personalized relaxation guidance.
Built for students, professionals, and anyone looking to optimize their rest through clean UI and actionable data.

Features

Sleep Tracking
Log daily sleep duration, quality, and habits.

Analytics Dashboard
Visual insights into patterns, trends, and weekly cycles.

Smart Suggestions
Personalized relaxation techniques, tips, and routines based on sleep behavior.

Wellness Tools
Breathing exercises, focus timers, light/sound recommendations, and more.

Responsive UI
Smooth, clean, and minimal interface designed for modern devices.

1. Clone the repo

cd SleepSync

2. Install dependencies
npm install

3. Run the development server
npm run dev
